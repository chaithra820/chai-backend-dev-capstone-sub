Little Lemon API Project

This project is built using Django and Django REST Framework.
It provides APIs for managing menu items and table bookings for the Little Lemon restaurant.

API Endpoints:

Home page:
/

Menu:
GET, POST, PUT, DELETE
/api/menu/

Bookings:
GET, POST, PUT, DELETE
/api/bookings/

Authentication:
Token authentication
/api/api-token-auth/

/auth/users/
/auth/token/login/

The APIs can be tested using the Django browsable API or the Insomnia REST client.
